Algarítmo ("receita de bolo")
Exemplo: Criar um app para celular a média de duas notas

Como calcular a média
        nota 1 + nota 2
Média = ________________
                2


// Variáveis
3 -> numéricas (nota1,nota2,media)

// Entrada
nota1, notas

//Processamento
media = (nota1 + nota2) / 2

// Saída (é o resultado)
media
-------------------------

Variáveis do mesmo propósito podem ser criadas em uma única linha separada por vírgula
