um App que informe qual é o combustível mais vantajoso ( etanol x gasolina) para
abastecer um carro flex

O etanol vale a pena quando custar até 70% do valor da gasolina.

//Variáveis
etanol, gasolina (numéricas)
//Entrada
etanol, gasolina
//Processamento/Saída
Se o preço do litro do etanol custar até 70% do preço do litro da gasolina
abastecer com etanol
senão
abastecer com gasolina

if (etanol < 0.7 * gasolina) {
    //abastecer com etanol
} else {
    //abastecer com gasolina
}
